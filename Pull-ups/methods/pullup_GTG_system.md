# Grease-the-Groove (GTG) Pull-up System

## Core Principle

GTG focuses on:

-   Neural efficiency
-   Movement familiarity
-   Motor unit recruitment

Rule:

Never train to fatigue.

## GTG Parameters

Current max = 6

GTG set size:

2--3 reps per set

Never exceed:

3 reps

## Example Daily Schedule

8:00 → 3 pull-ups\
9:00 → 3 pull-ups\
10:30 → 3 pull-ups\
13:00 → 3 pull-ups\
15:00 → 3 pull-ups\
18:00 → 3 pull-ups

Total daily volume:

18 reps

## Integration with Training

GTG does not replace main training.

Example schedule:

Monday -- training + 3--4 GTG sets\
Wednesday -- training + 3--4 GTG sets\
Friday -- training + 3--4 GTG sets

Tuesday / Thursday:

5--6 GTG sets

Sunday (test day):

No GTG.

## Weekly GTG Volume

Example:

5 sets × 3 reps × 7 days

≈105 reps

Low fatigue because intensity is low.

## When to Increase Reps

When 3 reps becomes very easy:

Upgrade to 4 reps.

Usually corresponds to:

Max ≈ 8 pull-ups.

## Common Mistakes

1.  Training to fatigue\
2.  Doing many sets in a row\
3.  Testing max every day

Correct rule:

Rest ≥ 30 minutes between sets.
