# Core Training Manual v3

Daily Core System for **L-Sit + Pull-up Stability**

------------------------------------------------------------------------

# Overview

Goal:

-   Unlock **L-sit**
-   Improve **pull-up body rigidity**
-   Build **gymnastics-style core strength**

Training frequency

-   **2 sessions per day**
-   **15--20 minutes total**

Structure

Home Session → Core activation + compression\
Gym Session → Support + L-sit skill

------------------------------------------------------------------------

# HOME SESSION (8--10 min)

## 1 Dead Bug

![Dead
Bug](../images/deadBug_001.jpg)
![Dead
Bug](../images/deadBug_002.jpg)

Purpose

-   Activate deep core
-   Stabilize pelvis
-   Protect lower back

Execution

1.  Lie on floor
2.  Press lower back into ground
3.  Extend opposite arm and leg

Tempo

2 sec extend\
1 sec pause\
2 sec return

Breathing

Exhale during extension\
Inhale during return

Volume

- 4 reps × 3 sets  
- Tempo: 2s extend / 1s hold / 2s return  
- Rest: 30s  

------------------------------------------------------------------------

## 2 Hollow Hold
![Hollow
Hold](../images/HollowBody_002.jpg)

Purpose

-   Anti‑extension core
-   Pull‑up body stability
-   Gymnastics core foundation

Execution cues

-   Lower back pressed to floor
-   Pelvis slightly posterior tilt
-   Rib cage down

Breathing

Small relaxed breaths

Volume
- 20s × 3 sets  
- Rest: 45s  

Progression goal

45 sec hold

------------------------------------------------------------------------

## 3 Seated Compression Lift

![Compression
Lift](../images/l-sit_005.jpg)

Purpose

-   Key strength for **L-sit**
-   Hip flexor development
-   Active leg compression

Setup

Sit upright\
Legs straight\
Hands beside hips\
Lean slightly forward

Movement

Lift both legs simultaneously

Even **1--5 cm lift counts**.

Regression

Single‑leg lifts if necessary

Technique cues

Knees locked\
Toes pointed\
Pause 1 sec at top

Breathing

Exhale while lifting\
Inhale while lowering

Volume
- 8 reps × 4 sets  
- Rest: 60s  
- Execution: lift → hold 1s → slow lower (2–3s)  

------------------------------------------------------------------------

## 4 Compression Hold（坐姿双腿压缩保持）

![Compression
Hold](../images/compressionHold_002.jpg)

动作要领
-   坐姿直腿，膝盖锁死，脚尖绷直
-   双手放在髋旁（屁股两侧）
-   手用力向下压地
-   下腹收紧，把双腿抬离地面
-   保持 1–3 秒


Movement
-   抬腿
-   停住
-   慢慢放

关键感觉
-   下腹发力
-   大腿根发力
-   手在用力压地

避免：
-   身体后仰
-   膝盖弯
-   甩腿

目标：接近解锁 L-sit
-   双腿离地
-   保持 2–3 秒

Volume
- 5 reps × 3 sets  
- Rest: 60s  
- Target: 2–3 seconds hold 

------------------------------------------------------------------------

# GYM SESSION (After Pull-up Training)

## 1 Support Hold

![Support
Hold](../images/supportHold.jpg)

Purpose

-   Shoulder depression strength
-   Straight arm support
-   L‑sit foundation

Execution

Lock elbows\
Push shoulders down\
Lift body tall

Cue

**Push the bars down**

Breathing

Normal breathing

Volume
- 20s × 4 sets  
- Rest: 45s  

------------------------------------------------------------------------

## 2 Tuck L‑Sit

![Tuck
L-Sit](../images/tuck-L-sit_001.jpg)


Purpose

-   Integrate core compression with support
-   Direct L‑sit progression

Execution

Push up on bars\
Bring knees to chest\
Feet off floor

Technique cues

Shoulders depressed\
Elbows locked\
Knees tight to chest

Breathing

Small breaths

Volume
- 10–12s × 4 sets  
- Rest: 60s  
- Goal: 15s  

------------------------------------------------------------------------
## 4 L-sit Negative
Volume
- 5 reps × 3 sets  
- Execution:
  - tuck → extend → return  
  OR  
  - lift → slow lower  

## 5 L‑Sit Attempts

Attempt full L‑sit at the end of training.

Even minimal lift counts.

Volume
- 3–5 attempts  
- NOT main training

# Key Technique Rules

Always remember

-   Shoulders depressed
-   Elbows locked
-   Slight forward lean

Mental cue

Push bars down\
Lean forward

This makes lifting legs easier.

------------------------------------------------------------------------

# Execution Rules

Train **2 sessions daily**\
Never train to failure\
Keep movements controlled\
Prioritize technique

Consistent training usually unlocks **L‑sit in 3--4 weeks**.
