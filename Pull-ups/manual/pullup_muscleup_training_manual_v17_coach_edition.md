# **一、当前训练状态**

* Strict Pull-up ≈ 5（第6个接近完成）  
* Dip ≈ 12  
* Toes-to-bar ≈ 5–8  
* 当前阶段：Week 5  
* 短期目标：Strict Pull-up ≥ 8  
* 长期目标：Muscle-up

| 教练提示：当前阶段的核心任务不是追求花样，而是把 Pull-up 从 5–6 个稳定推进到 8 个以上，同时为未来的 Muscle-up 建立顶部力量、爆发拉力和核心张力。 |
| :---- |

# **二、能力地图与三大能力系统**

## **1）三大能力系统**

| 系统 | 能力路径 |
| :---- | :---- |
| Pull 系统 | Active Hang → Scapular Pull → Pull-up 5 → Pull-up 8 → Chest-to-Bar → High Pull → Transition → Muscle-up |
| Push 系统 | Push-up → Dip → Muscle-up Lockout |
| Core 系统 | Hollow Hold → L-sit → Hanging L-sit → Toes-to-bar |

## **2）Pull-up 力量结构**

肩胛控制 \+ 核心稳定

**↓**

背阔肌基础力量

**↓**

力量耐力

**↓**

爆发拉力

**↓**

顶端控制

## **3）Muscle-up 能力门槛**

* Pull-up ≥ 10  
* Dip ≥ 15  
* Chest-to-bar ≥ 5

| 教练提示：Muscle-up 不是单一动作，而是 Pull strength、Explosive pull、Transition strength、Dip strength 四个模块的组合。 |
| :---- |

# **三、关键技术标准**

## **1）Pull-up 技术六步骤**

* 1\. Active Hang  
* 2\. 肩胛下沉  
* 3\. 背阔肌启动  
* 4\. 肘向下拉  
* 5\. 胸靠近杠（不是下巴找杠）  
* 6\. 控制下降（2–3 秒）

## **2）Muscle-up Transition 技术**

* 胸贴杠  
* 身体前倾  
* 手腕翻转  
* 肘转到杠上方  
* 训练：低杠 drill \+ 弹力带 muscle-up 3×3

## **3）Dip 技术节奏**

* 2 秒下降  
* 底部停 2 秒  
* 控制推起  
* 顶峰停 1 秒

## **4）L-sit 技术与进阶**

* Tuck L-sit → Single-leg → Full L-sit → 30 秒 L-sit

# **四、每周训练结构**

| 日期 | 训练主题 |
| :---- | :---- |
| 周日 | Pull-up 测试 |
| 周一 | Rest / Mobility |
| 周二 | CrossFit |
| 周三 | Pull-up 主训练 |
| 周四 | Core / 技术 |
| 周五 | CrossFit |
| 周六 | Rest |

| 教练提示：不同训练日的热身不一样。Pull-up 主训练日重在背部激活；Core / 技术日重在核心与肩胛；测试日重在神经唤醒但不能产生疲劳。 |

# **五、周三 Pull-up 主训练日**

**训练目标：**提高 Pull-up 基础力量、爆发拉力与顶部控制，是本手册最关键的训练日。

## **Warm-up**

* 肩部环绕 20×2  
* Band pull-aparts 20×2  
* Scapular Pull-up 3×6  
* Active Hang 3×15s  
* Hollow Hold 2×20s

## **Main Training**

* Pull-up 4×6  
* Explosive Pull-up 4×2  
* Slow Negative Pull-up 3×2（5秒下降）  
* Dip 4×10（底部停2秒，顶峰停1秒）  
* L-sit 3×15秒

## **Core / 补充**

* Top Hold 2×5–10秒（如顶部薄弱）  
* Upper-range Pull-up 2×6（从眼睛高度拉到下巴过杠）  
* False Grip Hang 20秒 ×3（为未来 Muscle-up 适应握法）  
* Hollow Body Hold 20–30秒 ×3（用于核心张力补充）

## **练后拉伸**

* Lat stretch 30秒 ×2  
* Chest stretch 30秒 ×2  
* Rear shoulder stretch 30秒 ×2  
* Forearm stretch 30秒 ×2

| 教练提示：当第4个开始明显变慢，通常说明当前瓶颈是背部耐力而不是单纯技术。优先把总量做稳，不要每组都拉到极限。 |
| :---- |

# **六、周四 Core / 技术日**

**训练目标：**强化核心、肩胛控制与技术质量，不追求疲劳，不破坏恢复。

## **Warm-up**

* Active Hang 2×15秒  
* Scapular Pull-up 2×6  
* Band pull-aparts 2×20  
* Hollow Hold 2×20秒

## **Main Training**

* Hollow Hang 3×15秒  
* Scapular Pull-up 3×6  
* L-sit 3×10–15秒  
* Australian Row 3×10  
* Top Hold 2×5–10秒（可选）

## **Core / 补充**

* Hollow Body Hold 20–30秒 ×3  
* Dead Bug 2×10（如腰部容易代偿）

## **练后拉伸**

* Lat stretch 30秒 ×2  
* Chest stretch 30秒 ×2  
* Rear shoulder stretch 30秒 ×2  
* Forearm stretch 30秒 ×2

| 教练提示：这一天的任务不是练废自己，而是把动作做干净。若前一天上背和大臂疲劳明显，可减少 20–30% 总量。 |
| :---- |

# **七、周日 Pull-up 测试日**

**训练目标：**测试当前 Strict Pull-up 水平，同时保留少量高质量补充训练。

## **Warm-up**

* Active Hang 2×10秒  
* Scapular Pull-up 2×4  
* 轻 Pull-up 2×2  
* 如状态较冷，可加 Band Assisted Pull-up 2×3  
* 休息 2–3 分钟后开始测试

## **Main Training**

* Pull-up Max Test  
* Back-off：2 × (max − 2\)  
* Explosive Pull-up 3×1  
* L-sit 3×10–15秒  
* Slow Negative 1–2次

## **Core / 补充**

* 若第6个接近过杠但需要踢腿：成绩记录为 5 个严格引体  
* 随后补做 Top Hold 2–3组 ×5–10秒  
* Slow Negative Pull-up 2–3组 ×2次，每次 3–5 秒下降

## **练后拉伸**

* Lat stretch 30秒 ×2  
* Chest stretch 30秒 ×2  
* Rear shoulder stretch 30秒 ×2  
* Forearm stretch 30秒 ×2

| 教练提示：测试当天不要做太多热身 Pull-up。目标是神经唤醒，而不是提前消耗背部。 |
| :---- |

# **八、周一 Rest / Mobility 日**

**训练目标：**恢复肩背、维持肩胛活动，帮助后续训练质量。

## **Warm-up**

* 轻步行 5–10 分钟（可选）

## **Main Training**

* 肩关节活动 2–3分钟  
* Band pull-aparts 2×20  
* 背阔肌泡沫轴 2–3分钟  
* Active Hang 2×15秒  
* 胸椎拉伸 30秒 ×2

## **练后拉伸**

* 如上背或前臂紧，可加 Forearm stretch 30秒 ×2  
* 如胸口紧，可加 Chest stretch 30秒 ×2

| 教练提示：这是恢复日，不是补课日。感觉状态好也不要擅自加 Pull-up 总量。 |
| :---- |

# **九、CrossFit 日（周二 / 周五）**

**训练目标：**保留 CrossFit 参与感，但避免破坏 Pull-up 专项进度和膝盖康复。

## **Warm-up**

* Band pull-aparts 2×15  
* Shoulder circles 20次

## **Main Training**

* 正常参加课堂，但优先避开或缩减大量 Toes-to-bar、大量 Pull-up、Muscle-up、Rope climb  
* 如当日 WOD 对背部和握力消耗很大，应主动降量或改动作  
* 若有 Pull-up 相关内容，优先保持技术，不追求刷数量

## **Core / 补充**

* 膝盖方面暂时优先 Hip-dominant 动作；避免深蹲、跑跳、深膝角高容量

## **练后拉伸**

* Lat stretch 30秒 ×2  
* Chest stretch 30秒 ×2  
* Rear shoulder stretch 30秒 ×2  
* Forearm stretch 30秒 ×2

| 教练提示：专项目标是 Pull-up 进步，因此 CF 现在属于辅助系统，而不是主系统。 |
| :---- |

# **十、安全训练总量与弹力带规则**

## **1）Pull-up 每周安全训练总量**

* 推荐周总量：40–60  
* 超过 80：肘部受伤风险增加

## **2）弹力带使用规则**

* 优先自重 Pull-up  
* 动作变形时用轻弹力带补足次数  
* 弹力带用于保护技术

# **十一、Week5 → Week8 进阶**

* Week5：4×6  
* Week6：4×6  
* Week7：3×7  
* Week8：Max Test ≥8

| 教练提示：如果某一周疲劳明显、睡眠变差、CF 背部量偏大，可以保守地重复上一周，不必强行进阶。 |

Pull-up 测试后的补充训练（新增整合内容）  
【新增规则】当测试中出现“第6个下巴接近过杠但需要踢腿”的情况：  
1）测试成绩记录为 5 个严格引体，不计算借力完成的重复。  
2）测试后立即进行弱点强化训练：  
   \- Top Hold（下巴过杠保持）2–3 组 × 5–10 秒。  
   \- Slow Negative Pull-up（慢离心）2–3 组 × 2 次，每次 3–5 秒下降。  
3）目的：强化最后 5–10 cm 的顶部力量，解决“差一点过杠”的瓶颈。

Chris 方法整合（顶部强化部分）  
【新增动作整合】从 Chris Heria 训练方法中吸收的关键内容：  
• Top Hold：跳到下巴过杠位置保持 5–10 秒，强化顶部力量。  
• Upper-range Pull-up：从眼睛高度拉到下巴过杠，2 组 × 6 次，用于强化中段到顶部。  
• False Grip Hang：假握悬挂 20 秒 × 3 组，用于为未来的 Muscle-up 做握法适应。  
说明：这些动作只作为补充训练，不替代原有 Pull-up 主训练结构。
