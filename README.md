
# Calisthenics Training System

This repository contains a structured training system focused on:

- Pull-up strength
- L-sit progression
- Core strength
- Future muscle-up development

## Training Philosophy

- High frequency skill training
- Do not train to failure
- Focus on technique quality
- Build progressive strength

## Modules

Pull-ups → strength foundation  
Core → L-sit support system  
L-sit → static skill progression  
Muscle-up → advanced pulling skill
